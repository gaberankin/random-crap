It's a fairly simple jquery plugin, not much to it aside from 3 files (2 images and the jquery.doge.js javascript file), not including the jquery requirement:

<script type="text/javascript" src="//code.jquery.com/jquery-1.11.0.min.js"></script>
<script type="text/javascript" src="javascripts/doge/jquery.doge.min.js"></script>

The code for the above demo is thus:

$(document).ready(function(){
  $('#doge1').doge({
    imagePath: 'javascripts/doge',
    begin: function(){
      $('#doge1 h3').data('original-text', $('#doge1 h3').text()).text('much comedy');
    },
    complete: function(){
      $('#doge1 h3').text($('#doge1 h3').data('original-text'));
    }
  });
});

The main thing to be concerned with is to make sure that the plugin can find the doge images, which is what the imagePath option is for. This path should be relative to the page that is calling the plugin.

The begin and complete options are callbacks that occur at the start and end of the doge animations. What I'm doing here is changing the text of the button while the event occurs, and reverting it after it's complete.

There are two more options that I'm not using here in the demo: enterOn (either 'click' or 'timer'. 'click' is the default) and delayTime (represented in milliseconds, is the length of time that the timer should run before the doge animation begins after the plugin code is run). These two options can be used for surprises.
